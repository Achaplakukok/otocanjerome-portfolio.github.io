Project Overview

About the Game

CodeQuest Maze is an educational mobile game tied to IT lessons that helps students recall basic IT topics through engaging, interactive gameplay. Players navigate progressively challenging levels while reinforcing foundational concepts.

The Challenge

Students often forget IT basics when lessons feel abstract. Traditional learning can lack the engagement needed to keep recall strong.

The Solution

CodeQuest Maze gamifies IT lessons by combining puzzle-solving with interactive tasks. Players use simple commands to guide their character through mazes, receiving immediate feedback and rewards that reinforce key topics.
